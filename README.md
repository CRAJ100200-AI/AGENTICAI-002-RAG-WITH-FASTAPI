# HR RAG Agent

A simple Retrieval-Augmented Generation (RAG) system using LangChain + Gemini.

## Features
- Vector search using FAISS
- Gemini embeddings + LLM
- FastAPI interface
- Runnable in Google Colab

## Run (Colab)
1. Install requirements
2. Set GOOGLE_API_KEY
3. Start FastAPI
4. Expose using ngrok
